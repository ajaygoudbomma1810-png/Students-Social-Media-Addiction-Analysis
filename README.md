# Students Social Media Addiction Analysis  
### Python Data Cleaning + Power BI Dashboard

This project analyzes the impact of **social media addiction among students** using a full data analytics pipeline built with **Python** and **Power BI**.

The goal is to understand usage patterns, addiction levels, and how excessive social media impacts **sleep, academics, mental health, and lifestyle**.

##  Project Overview

This project provides insights into:

- Average daily usage of social media  
- Addiction score calculation  
- Impact of social media on:
  - Sleep  
  - Productivity  
  - Academic performance  
  - Stress/Anxiety levels  
- Platform-wise addiction (Instagram, Snapchat, YouTube, etc.)

The cleaned dataset is exported via Python and imported into Power BI for creating an interactive dashboard.

---

##  Data Cleaning Steps (Python)

The pipeline includes:

- Handling missing values  
- Fixing inconsistent data  
- Converting data types  
- Creating new features (e.g., **Addiction_Score**)  
- Encoding categorical variables  
- Exporting a clean, ready-to-use CSV for Power BI

##  Key Insights

- Students using social media for **4+ hours/day** showed reduced sleep duration  
- High addiction levels correlate with **lower academic performance**  
- Instagram & Snapchat users displayed **higher anxiety indicators**
